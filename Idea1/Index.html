Speech Emotion Recognition using Machine Learning

Abstract:-

Emotion recognition is the part of speech recognition which is gaining more popularity and need for it increases enormously. Identifying emotion from speech is a nontrivial task pertaining to the ambiguous definition of emotion itself. In this work, we adopt a feature engineering based approach to tackle the task of speech emotion recognition. We are using machine learning techniques, to recognize emotion and classify the emotion according to the speech signals.
